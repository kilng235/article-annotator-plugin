# Article Annotator

An Obsidian plugin for highlighting text and adding annotations, similar to Microsoft Word comments.

## Features

- Highlight selected text with multiple colors
- Add notes to highlighted text
- Browse annotations in a sidebar
- Search all annotations
- Export annotations from the current file
- Drag to reorder annotations
- Mobile floating action button support

## Installation

### Community Plugins
1. Open Obsidian Settings.
2. Go to Community plugins.
3. Search for `Article Annotator`.
4. Install and enable it.

### Manual Installation
1. Download `article-annotator.zip` from the latest GitHub Release.
2. Extract it into `.obsidian/plugins/article-annotator/`.
3. Make sure the folder contains `main.js`, `manifest.json`, and `styles.css`.
4. Enable the plugin in Obsidian.

## Notes

- Current stable support is focused on Markdown files.
- `data.json` is user data and is not included in the release package.
- Backup your vault regularly.

## Release Assets

Each release provides:
- `article-annotator.zip`
- `main.js`
- `manifest.json`
- `styles.css`
